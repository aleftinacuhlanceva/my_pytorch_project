# Image Processing App

Десктоп-приложение для обработки изображений, разработанное на языке Python с использованием библиотеки OpenCV.

## Используемое окружение

- Python 3.9.13
- PyTorch 1.11.0 (CPU)
- OpenCV 4.9.0
- NumPy 1.26.4
- Pillow 11.3.0

## Возможности

- загрузка изображений PNG/JPG;
- получение снимка с веб-камеры;
- отображение выбранного изображения в окне приложения;
- просмотр красного, зеленого или синего канала;
- выделение пикселей, у которых интенсивность красного цвета выше заданного порога;
- повышение резкости изображения;
- рисование синего прямоугольника по координатам пользователя;
- сохранение результата.

## Запуск приложения

1. Активировать окружение:

```powershell
.\torch_venv_py39\Scripts\Activate.ps1
```

2. Установить необходимые зависимости:

```powershell
pip install -r requirements.txt
```

3. Запустить приложение:

```powershell
python app.py
```

## Возможные проблемы при работе с веб-камерой

Если приложению не удалось получить изображение с веб-камеры, рекомендуется проверить:

- подключение камеры;
- наличие драйверов;
- разрешение на использование камеры в Windows;
- не используется ли камера другим приложением.
