# Image Processing App

Настольное приложение для обработки изображений, разработанное на языке Python с использованием библиотек OpenCV, Tkinter, Pillow и NumPy.

## Возможности приложения

Приложение позволяет:

* загружать изображения форматов JPG и PNG;
* получать изображение с веб-камеры;
* отображать красный, зелёный и синий цветовые каналы изображения;
* выделять объекты красного цвета;
* повышать резкость изображения;
* рисовать синий прямоугольник по координатам, введённым пользователем;
* сохранять результат обработки изображения.

## Системные требования

* Python 3.9 или выше;
* библиотека OpenCV;
* библиотека Pillow;
* библиотека NumPy;
* библиотека PyTorch (при необходимости использования окружения, подготовленного в первой части практики).

## Инструкция по сборке, запуску и использованию приложения

### 1. Клонирование репозитория

```bash
git clone https://github.com/aleftinacuhlanceva/my_pytorch_project.git
cd my_pytorch_project
```

### 2. Создание виртуального окружения

**Windows**

```powershell
python -m venv venv
.\venv\Scripts\Activate.ps1
```

**Linux / macOS**

```bash
python3 -m venv venv
source venv/bin/activate
```

### 3. Установка зависимостей

```bash
pip install -r requirements.txt
```

### 4. Запуск приложения

```bash
python app.py
```

После выполнения команды откроется главное окно программы.

## Пример использования

1. Запустить приложение.
2. Нажать кнопку **«Загрузить изображение»** и выбрать изображение формата JPG или PNG.
3. При необходимости можно получить изображение с веб-камеры с помощью соответствующей кнопки.
4. Выбрать одну из функций обработки:

   * отображение красного, зелёного или синего канала;
   * выделение красных объектов;
   * повышение резкости изображения;
   * рисование синего прямоугольника.
5. Просмотреть результат обработки.
6. При необходимости нажать кнопку **«Сохранить результат»** и сохранить обработанное изображение.

## Возможные проблемы

### Не удаётся подключиться к веб-камере

Возможные причины:

* веб-камера используется другим приложением;
* отсутствует разрешение на использование камеры;
* камера не подключена или отключена в системе.

Рекомендуется закрыть другие программы, использующие камеру, проверить настройки конфиденциальности операционной системы и повторить запуск приложения.
